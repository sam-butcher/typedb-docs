= `@cascade` annotation

The `@cascade` annotation is used
// tag::overview[]
to specify behaviour when deleting a relation's role players.
// end::overview[]

== Syntax

.Syntax
[,typeql]
----
<type label> relates <role type label> @cascade;
----

The `@cascade` annotation does not accept any arguments.

== Usage

[NOTE]
====
The `@cascade` annotation is not yet available in TypeDB.
Coming soon!
====
