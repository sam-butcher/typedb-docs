= Operators
:page-aliases: {page-version}@new_reference::typeql/pages/values/arithmetic.adoc, {page-version}@new_reference::typeql/pages/values/functions.adoc

Arithmetic operator expressions allow computation of new values. Attributes are treated as pure values when used in an arithmetic operator
expression.

== Quick reference

[%header,cols="^1,^1,5"]
|===
|Precedence
|Operator
|Name

|1 |`^` |exponentiation (power)
|2 |`*` |multiplication
|2 |`/` |division
|2 |`%` |modulo (remainder)
|3 |`+` |addition
|3 |`-` |subtraction
|===

Parentheses (`()`) may be used to override the precedence. The sub-expressions in parentheses are always evaluated first.

[,typeql]
----
(1 + 2) * 3 == 9
----
